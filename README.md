# HabitFlow | Modern Habit Tracker

A production-ready Habit Tracker built with Next.js 16, Tailwind CSS 4, and Clerk Authentication.

## Features
- **Interactive Habit Tracking**: 14-day scrollable window with historical navigation.
- **Advanced Analytics**: Global distribution pie charts and individual monthly mastery tracking.
- **Secure Auth**: Google Login via Clerk with private route protection.
- **Glassmorphic UI**: Premium dark-themed design with vibrant charts.

## Deployment
This project is configured for seamless deployment to [Vercel](https://vercel.com).
Copy `.env.local` variables to your Vercel Project settings.
